8x8 Multiplier in Chisel

8x8 Multiplier Core architecture is detalied in Mult8x8.doc file .

Install Java
Install SBT

Getting the Repo and running 
----------------------------

    $ git clone https://github.com/jardelufc/chisel-mult8x8.git
    $ cd chisel-mult8x8
    $ git fetch origin
    $ git checkout release
    $ ./run-mult8x8 mult8x8
